# User Authentication System Upgrade Guide

## What's New
We've upgraded the Family Wishlist application with a new user authentication system! This brings several improvements:

1. **Individual Accounts**: Each family member now has their own username and password
2. **Direct Access**: Log in directly to your wishlist without needing to select your user each time
3. **Password Recovery**: Reset your password via email if you forget it
4. **Enhanced Security**: More robust password storage and authentication

## For Existing Users
For a smooth transition, temporary credentials have been created for all existing accounts:

- **Username**: Your name with spaces replaced by underscores (e.g., "jane_smith")
- **Password**: Your name without spaces followed by "123" (e.g., "JaneSmith123")

The Admin account username is simply "admin" and uses either the previous family password or a default password.

**Important**: Please change your password after logging in for the first time!

## Using the New System

### First Login
1. Visit the login page
2. Enter your username and password 
3. You'll be taken directly to your wishlist

### Changing Your Password
We recommend changing your password after first login:
1. Click on your profile menu in the top right corner
2. Select "Change Password"
3. Enter your current password and your new password
4. Submit the form

### If You Forget Your Password
1. Click the "Forgot Password" link on the login screen
2. Enter your username or email address
3. Follow the reset link sent to your email
4. Create a new password

### Legacy Login 
The old family password still works during the transition period. Just click "Use legacy family password" on the login screen.

## Questions or Issues?
Please contact the family administrator for help with any login issues or questions about the new system.
